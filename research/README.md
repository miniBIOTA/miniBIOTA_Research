# Research Notes

Use this folder for working research notes. Stable reusable workflows live in `skills/`, exact reusable references and templates live in `skills/*/reference/`, and archived docs/templates are historical only.

Recommended structure:

| Folder | Purpose |
|---|---|
| `species/` | Source-backed species research |
| `ecological-mechanisms/` | Reusable explanations of ecological processes |
| `experiments/` | Background research for future experiments |
| `identification-notes/` | Organism ID notes and confidence tracking |
| `claim-checks/` | Verification notes before claims become public |
| `source-library/` | Source notes, citations, and reference summaries |
| `open-questions/` | Unresolved questions and evidence gaps |
| `research-briefs/` | Handoff briefs for Content, Brain, Website, or other agents |

Keep notes source-backed and uncertainty-labeled. When research is meant for `miniBIOTA_Content`, create a brief in `research-briefs/` rather than drafting a final script or polished public asset.
